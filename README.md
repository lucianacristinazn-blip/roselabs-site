# Rose Labs — Site Institucional

Site da Rose Labs — Arquitetura de Crescimento Sustentável.

## Stack
- **Framework**: Astro
- **CSS**: Tailwind CSS + CSS customizado
- **CMS**: Keystatic (a adicionar)
- **Deploy**: Cloudflare Pages
- **Repo**: GitHub

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

1. Push para GitHub
2. Cloudflare Pages detecta automaticamente e faz o deploy

## Estrutura de páginas

- `/` — Home (landing/sales page)
- `/sobre` — Quem Somos, Fundadora, Manifesto
- `/metodologia` — Metodologia e Framework
- `/servicos` — Serviços e preços
- `/confiabilidade` — Confidencialidade + Carta aos Líderes
- `/blog` — Blog (CMS via Keystatic)
- `/contato` — Formulário de contato
